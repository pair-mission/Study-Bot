# 프로젝트 개요

스터디의 모임 일정을 관리하고, 참여자들에게 리마인드 및 출석을 안내하는 콘솔 기반 애플리케이션입니다.

# 기능 목록

## 모임 관리

- 모임 등록
    - [x] 스터디 모임의 날짜, 시작 및 종료 시간, 주제, 장소를 입력받아 모임을 등록한다.
    - 예외 상황
        - [x] 모임의 날짜, 시작 및 종료 시간, 주제, 장소 중 정보가 하나라도 없으면 예외가 발생한다.
        - [x] 모임의 날짜가 현재 기준으로 과거인 경우 예외가 발생한다.
        - [x] 날짜의 형식을 "YYYY-MM-dd"로 입력하지 않으면 예외가 발생한다.
        - [x] 날짜 입력값이 빈 문자열이거나 공백이면 예외가 발생한다.
        - [x] 모임의 종료 시간이 시작 시간보다 빠른 경우 예외가 발생한다.
        - [x] 시간의 형식을 "HH:mm"으로 입력하지 않으면 예외가 발생한다.
        - [ ] 모임의 시간은 30분 단위로 입력 가능하며, 24시 기준으로 입력해야한다. (오후 6시는 18:00)
        - [ ] 모임의 주제나 장소가 빈 문자열이거나 공백이면 예외가 발생한다.
